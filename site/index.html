à propos

c'est sur le côté...

Faire une application Web avec nodejs est plus simple qu'il n'y paraît! En fait, je dirais que c'est plus simple qu'en PHP! On lance l'application avec un : 

cd server
nodejs index.js

description

Le point d'entrée, c'est index.js. Voici la description des répertoires:

  • server: le code du serveur,
  • site: le contenu du site.

Les principaux fichiers du code:

  • index.js: le point d'entrée,
  • configuration.js: la gestion de la configuration,
  • server.js: la gestion du serveur,
  • router.js: la gestion du routage.

Le fichier de configuration est config/mainconfig.json.

c'est cool, non?
